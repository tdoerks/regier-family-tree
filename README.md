# Regier Family Tree

Interactive family tree web application for the Regier family reunion spanning 4 generations.

## Features

- **Interactive Family Tree** - Expandable/collapsible family branches with color-coded family lines
- **Search & Filter** - Find family members by name, location, generation, or family line
- **Family Bingo** - Interactive reunion game with 24 fun challenges
- **Reunion Passport** - Collect stamps by meeting members from each family line
- **Relationship Quiz** - Test your knowledge of family connections
- **Statistics Dashboard** - View family demographics at a glance
- **Who's Who Table** - Complete sortable list of all family members

## Password Protection

This site is for Regier family members only. Access requires the family password.

**Note:** The password protection is JavaScript-based (client-side) and is meant to deter casual visitors, not provide strong security. The site contains only names and relationships - no sensitive personal information.

## Privacy

- Site is blocked from search engines via robots.txt
- No personal data (addresses, phone numbers, exact birthdates) is stored
- Only family names, relationships, and general locations (city/state)
- Game progress stored locally in your browser only

## Technology

- Pure HTML/CSS/JavaScript (no frameworks)
- Responsive design for mobile and desktop
- LocalStorage for password persistence and game progress
- Print-friendly formatting

## Usage

1. Visit the site at: https://tdoerks.github.io/regier-family-tree
2. Enter the family password
3. Explore the family tree, play games, and connect with relatives!
